# user_editable_files

Manually-maintained data files that the pipeline reads exclusively from this folder.
All other data comes from SharePoint (step_00) or SpkData CSVs (main.py ETL).

**Copy or symlink the originals into this folder when setting up a new machine.**

## Files required

| File | Source | Description |
|------|--------|-------------|
| `merged_terminated_projects_testDetails_MetallurgicalBalance.xlsx` | db_python | Leaching performance for old terminated projects (01A, 002, 004, 006) |
| `dataset_characterization_summary_terminated_projects.xlsx` | db_python | Chemical characterisation for terminated projects |
| `dataset_mineralogy_summary_modals_terminated_projects.xlsx` | db_python | Mineralogy modals for terminated projects |
| `df_column_summary_terminated_projects.xlsx` | db_python | Column initial conditions for terminated projects |
| `dataset_reactor_summary_detailed_terminated_projects.xlsx` | db_python | Reactor summary for terminated projects |
| `df_qemscan_compilation.xlsx` | db_python | QEMSCAN mineral exposure data (user-maintained) |

## NOT in this folder (generated by the pipeline)

- `dataset_per_sample_reactor_complete.csv` — generated by step_03 and saved to
  `pipeline/outputs/`. Do NOT put it here manually.

## How to update

Replace the file in this folder with the updated version.
No code changes needed — just drop the new file in place.
